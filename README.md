# 🎮 Flappy Bird Clone (Unity)

A simple 2D Flappy Bird-style game built using Unity and C#.

---

## 🕹️ About the Game
This is a basic arcade-style game where the player controls a bird and tries to pass through pipes without colliding. The game focuses on simple physics-based movement and obstacle avoidance.

## ⚙️ Features
- Gravity-based player movement
- Tap/Click to make the bird jump
- Procedurally spawning pipes
- Score system based on survival time
- Simple restart system after game over

---

## 🛠️ Built With
- Unity Engine
- C#
- TextMeshPro (UI)

---

## 🎯 Controls
- Click / Tap → Jump
- Avoid pipes → Survive as long as possible

---

## 📌 Purpose
This project was made for learning Unity fundamentals including:
- 2D physics
- Game loops
- Prefab spawning
- UI and scoring system
